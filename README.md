## Dependencies

- Node.js
- exxpress
- ejs
- bcrypt
- cookie-parser
- cookie-session
- method-override

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run `npm install nodemon`, then start the server using npm start. (or run the development web server using the `node express_server.js` command.)
